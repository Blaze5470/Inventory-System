# Inventory and Purchase Order System

## Description
This program manages inventory, vendors, purchase orders, and shipments for an outdoor adventure supply company.

## Features
- Product management
- Vendor management
- Purchase order creation
- Shipment receiving
- Inventory reports
- JSON save/load system
- Reorder suggestion system

## Files
- main.py
- models.py
- inventory_manager.py
- file_manager.py
- reports.py

## Running the Program
Run the following command:

python main.py

## Data Storage
Data is stored in JSON files.

## Extra Feature
This program includes a reorder suggestion system.
